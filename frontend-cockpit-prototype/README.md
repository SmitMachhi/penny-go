# Penny operations cockpit prototype

Standalone frontend renovation prototype for option 3, the operations cockpit direction.

Open `index.html` directly in a browser. The prototype does not modify or import the active Svelte app code. It reuses Penny's existing colors, font families, and icon asset by referencing files already present under `web/`.

## Verification

```bash
node frontend-cockpit-prototype/smoke-test.mjs
```
